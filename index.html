How IPFS works?

When you add a file to IPFS, your file is split into smaller chunks,
cryptographically hashed, and given a unique fingerprint called a
content identifier (CID). This CID acts as a permanent record of your
file as it exists at that point in time. However, this doesn't mean you
need to remember a long string of CIDs to access files. Deupload
can be used to map CIDs to human-readable DNS names.
